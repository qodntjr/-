딥러닝은 기본 모델을 바탕으로,
프로젝트에 맞춰서 어떤 옵션을 더하고 어떤 층을 추가하느냐에 따라 성능이 좋아질 수 있음

컨볼루션 신경망(CNN)

입력된 이미지에서 다시 한번 특징을 추출하기 위해 마스크(필터,윈도 또는 커널)을 도입하는 기법
Conv2D( )
1 .첫 번쟤 인자 : 커널을 몇 개 적용할지 정함
2. kernel_size : 커널의 크기
3. inpurt_shape : Dense 층과 마찬가지로 맨 처음 층에는 입력되는 값을 알려주어야함
4. activation : 활성화 함수를 정의


폴링 또는 서브 샘플링
컨볼루션 층을 통해 이미지 특징을 도출
그 결과가 크고 복잡하면 다시 한번 축소 해야 함
맥스 폴링 : 최대값
평균 폴링 : 평균값

드롭아웃,플래튼
딥러닝 학습을 실행할 때 과적합을 얼마나 효과적으로 피해가는지 가장 중요하다
랜덤하게 노드를 끔으로써 학습 데이터에 지나치게 치우쳐서 학습되는 과적합을 방지 할 수 있다
Dropout( )

컨볼루션 층이나 맥스 풀링은 주어진 이미지를 2차원 배열인 채로 다루기 때문에
1차원 배열로 바꿔주어야 활성화 함수가 있는 층에서 사용할 수 있다
